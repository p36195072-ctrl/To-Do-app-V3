# ✅ To Do App V3

A beginner-friendly Python project that demonstrates how to build a To-Do application using Python, JSON, and file handling.

## 📌 Description

This project allows users to manage their daily tasks through a menu-driven command-line interface. Tasks are stored permanently in a JSON file, allowing users to access their tasks even after restarting the application.

## ✨ Features

* ➕ Add New Tasks
* 📋 View All Tasks
* ✅ Mark All Tasks as Completed
* ☑️ Mark Individual Tasks as Completed
* 🗑️ Delete Tasks
* 💾 Automatic JSON Saving
* 📂 Automatic JSON Loading
* 🚫 Prevent Duplicate Tasks

## 🧠 Concepts Used

* Dictionaries
* JSON
* File Handling

* CRUD Operations
